# V8のTorqueを語りたい

V8のbuiltinsを支える専用言語 **Torque** をテーマにしたSlidevスライドです。
CSA（CodeStubAssembler）の時代から、なぜTorqueが生まれ、V8の安全性と速度をどう支えているのかを紹介します。

Slidevで作成されています。
